## Changelog

All notable changes to this project will be documented in this file.

### [1.0.0] - 2024-01-13

#### Added
- Initial release of Social Media Platform
- User authentication with JWT
- Post creation with image support
- Comment system with emoji reactions
- User profiles with unique 11-digit code
- Like/unlike functionality for posts and comments
- Edit/delete for posts and comments
- Responsive UI with Tailwind CSS
- Docker containerization
- Comprehensive documentation

#### Features
- Multi-user support
- Real-time post feed
- Image uploads (base64)
- Editable content
- User profiles
- Emoji reactions on comments

#### Tech Stack
- Frontend: React 18, Vite, Tailwind CSS, Zustand
- Backend: Node.js, Express, MongoDB
- Authentication: JWT
- Deployment: Docker, Docker Compose
