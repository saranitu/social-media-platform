# Contributing to Social Media Platform

We welcome contributions! Please read these guidelines before submitting pull requests.

## Getting Started

1. Fork the repository
2. Clone your fork
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit: `git commit -am 'Add your feature'`
6. Push: `git push origin feature/your-feature-name`
7. Submit a Pull Request

## Code Style

- Use consistent indentation (2 spaces)
- Follow existing code patterns
- Add comments for complex logic
- Use meaningful variable names

## Testing

- Test locally before submitting PR
- Verify all endpoints work correctly
- Test on both desktop and mobile

## Issues

- Check existing issues before creating new ones
- Provide clear description and steps to reproduce
- Include error messages and screenshots if applicable

## Contact

For questions, reach out to the maintainers.

Thank you for contributing! 🙏
